Word Wide Web General

  1. What are the most essential technologies that made the World Wide Web such a success?
    Computer Networking, HTML, URL, HTTP.

  2. How are resources (html pages, images, etc) addressed in the web?
    They are adressed with URIs (uniform resource identifiers) which locate the address or information.

  3. What are the general components of a URL?
    • A scheme: identifies the protocol to access the resource on the internet
    • A host: identifies the host that holds the resource
    • A path: identifies the specific resource in the host that the web client wants to access
    • A query string: follows the path component and provides a string of information for the resource

Web Server

  1. What is a web server?
    It is a computer with/only Webserver-Software.

  2. What does a web server?
    A web server saves websites, process them an sends them to the clients (web servers). They communicate with HTTP.

  3. How can content be stored on a web server?
    A web server is a computer which means that the data can be saved normaly, it has to have a special software called client software to make the website accessable.

  4. How is content in webpages described?
    Content is described in form of all the information accessable in a website such as texts, links, images, audio, ...

Web Client

  1. What is a Web Client?
    A Web Client also named Browser and is used for sending requests to a Web Server.

  2. What does a web client?
    A Web Client is connecting to a Web Server to access the requested Website.

  3. Are Web Client and Web Browser the same?
    No they serve a completely different purpose. Web Clients send requests and Web Browsers provide information.
